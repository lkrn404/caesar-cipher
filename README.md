# Caesar Cipher

**WARNING:** this tool is for educational use only, and should **NOT** be used for securing sensitive information. It is **NOT** comparable to a modern encryption algorithm.

This program is a simple implementation of the [Caesar Cipher](https://en.wikipedia.org/wiki/Caesar_cipher) encryption algorithm in C++ for Linux, MacOS, and Windows via WSL.

Requirements:
- A UNIX or UNIX-like environment
- GCC or Clang
